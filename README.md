# LearnMate — AI Powered Study Companion 🎓

LearnMate is a web-based AI study companion for school students of **Classes 5 to 12**. It adapts to your class group and understanding level to deliver personalised explanations, doubt resolution, study notes, and practice quizzes — all powered by Google Gemini AI.

## 🌐 Live Demo
**[learnmate-ai-dusky.vercel.app](https://learnmate-ai-dusky.vercel.app)**

## ✨ What it does
- **Explain** any topic with simple language, analogies, and examples
- **Resolve doubts** with guided, step-by-step answers
- **Generate notes** in concise bullet-point format for quick revision
- **Create quizzes** with instant scoring and answer explanations

## 🛠️ Built With
React.js · Node.js · Express.js · MongoDB · Google Gemini API
